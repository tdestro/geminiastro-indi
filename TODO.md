# Future Work

No blocking production-readiness items remain for the tested GeminiAstro controller and Mak 180 rig. Hardware validation
results are recorded in [docs/hardware-validation-results-2026-06-21.md](docs/hardware-validation-results-2026-06-21.md).

Future work:

- Add an automated INDI hardware-test harness for repeatable bench validation.
- Test a true physical USB cable pull and physical focuser power interruption.
- Validate practical backlash behavior optically, not only by controller readback.
- Validate temperature-compensation behavior on a controller that reports compensation available.
- Test additional GeminiAstro controller variants before making claims beyond this hardware.
